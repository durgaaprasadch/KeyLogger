## 📌 Overview
This repository contains **two Python keylogger variants** plus a decryption tool:


### 1️⃣ **Keylogger-adv**
- 🔒 **KeyLoggerWithEncrypter.py** – Logs keystrokes encrypted using `cryptography.Fernet`
- 🔓 **decrypter.py** – Decrypts logs generated by the encrypted keylogger

### 2️⃣ 📝 **KeyLogger.py** – Logs keystrokes in plain text (no encryption)


  

Both variants support:
- ⏱ **Idle time detection** – Detects inactivity for a configurable time  
- 🕒 **IST timestamping** – Saves logs in Indian Standard Time  
- 🎯 **Custom stop hotkey** – Default: `CTRL+SHIFT+Q`  
- 📜 **Readable special key formatting** – e.g., `[ENTER]`, `[SPACE]`

---

## ✨ Features
- **Keystroke Logging** – Captures both presses & releases
- **Timestamping** – IST-based log entries
- **Idle Detection** – Notes inactivity periods over `IDLE_THRESHOLD` seconds
- **Special Key Handling** – User-friendly representation of keys
- **Encryption (Optional)** – `keyloggerWithEncrypter.py` encrypts logs with Fernet symmetric encryption
- **Safe Shutdown** – Stops cleanly via hotkey (default: `CTRL+SHIFT+Q`)

## ⚙️ Prerequisites
- **Python 3.x** installed
- Required libraries:
```bash
pip install keyboard cryptography
```
🚀 Usage


1️⃣ **Encrypted Keylogger**.
Run the script:

```
python keyloggerWithEncrypter.py
```
First run generates an encryption key (displayed in terminal) – save it securely.
Without it, you cannot decrypt the logs.

Logs are stored in /logs with timestamped filenames.
```
Press CTRL+SHIFT+Q to stop logging.
```
2️⃣ Decrypting the Logs
Open 
```
decrypter.py
``` 
And set:

```
ENCRYPTION_KEY = b'your-saved-encryption-key-here'
log_file = "C:/path/to/encrypted_keylog.txt"
output_file = "C:/path/to/Decrypted.txt"
```
Run:

```
python decrypter.py
```
The decrypted text is appended to the output file.

3️⃣Now the **Plain Text Keylogger**
Run:

```
python KeyLogger.py
```
Stop with CTRL+SHIFT+Q.

Logs are saved directly in plain text.

🛠 Configuration
Modify these in KeyLoggerWithEncrypter.py or KeyLogger.py:

IDLE_THRESHOLD = 60        # seconds before "IDLE" is logged

STOP_HOTKEY = 'ctrl+shift+q'  # stop logging


LOG_DIR = "logs"           # log storage folder

USE_ENCRYPTION = True      # only in encrypted version
